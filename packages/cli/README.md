# @react-scad/cli

Scaffold a new [react-scad](https://github.com/react-scad/react-scad) project. Built with [Brocli](https://github.com/drizzle-team/brocli) and TypeScript.

## Usage

Create a new project (canonical):

```bash
npx @react-scad/cli new my-project
```

Shorthand (project name as first argument):

```bash
npx @react-scad/cli my-project
```

Help and version:

```bash
npx @react-scad/cli --help
npx @react-scad/cli --version
```
